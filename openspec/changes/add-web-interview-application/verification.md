## Prototype verification

状态：前端合成数据原型已验证；真实 API 集成延后。

### Automated coverage

- 公开首页说明网页、Mac 伴随程序、AI 建议和隐私边界。
- 未授权访问不会渲染面试问题、回答或资料。
- 原型身份、应用首页、新建草稿和准备检查可以完成。
- 简历、JD 和知识库覆盖缺失、处理中、成功、失败、替换与删除状态。
- 未确认数据用途时无法开始面试。
- 实时工作台覆盖问题状态、回答依据、历史、手动输入去重、截图预览/失败/识别、暂停和结束。
- 复盘区分原始记录与 AI 摘要，并覆盖空记录、摘要失败、截图删除和整场删除。
- CSS 合约验证桌面三栏、平板抽屉、手机单栏、44px 触控目标、焦点样式和减少动态效果。
- 合成 AI 评测拒绝资料中不存在的公司、项目、职责与量化结果。
- 浏览器源码检查不得包含常见外部 AI 供应商密钥。

### Deferred behavior

- 当前 `FixtureInterviewAdapter` 只使用合成数据；面试草稿、资料解析、实时事件和复盘的真实 HTTP/实时 API 尚未提供。
- 文件选择只演示前端解析状态，不上传或持久化真实文件。
- 截图流程只使用合成预览，不调用真实 OCR/视觉模型。
- 音频区域消费协议形状与合成设备状态，不进行真实服务端转写。
- DMG 下载地址是产品入口，正式签名、公证与托管不属于本网页变更。

在真实服务端接口确定前，不应将上述延后行为标记为生产完成。
