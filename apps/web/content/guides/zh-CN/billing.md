# 积分、会员与支付

低频使用可以购买积分，高频面试可以选择 3、7、15 或 30 天会员。会员覆盖正常个人使用范围内的普通回答与截图；15 天和 30 天会员每个生效周期另含 2 份知识材料索引额度。

知识材料按照服务端 Token 报价：`max(200, 向上取整(Token / 1000) × 20)`。1–10,000 Token 为 200 点，10,001–11,000 Token 为 220 点。额度用完或使用 3/7 天会员时按此公式结算；失败、取消或相同用量重试不会重复扣点。

微信支付和支付宝通过官方订单完成。支付二维码绑定具体订单和金额并会过期。浏览器返回支付页后仍需等待服务端确认；未到账时不要重复付款，可刷新订单状态或联系客服。退款以订单状态和官方渠道结果为准。
