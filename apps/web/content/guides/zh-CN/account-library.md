# 微信账号与面试资料

微信登录通过官方授权窗口完成。扫码成功后系统创建或登录面试稳账号；如果微信已绑定其他账号，不会自动合并。

资料页分为“简历”“职位 JD”“知识库”。新增资料只进入可选资料列表，不会自动授权给任何面试；进入准备页后再决定本场使用什么，也可以全部不使用。

空知识库免费。添加 PDF、DOCX、TXT 或 Markdown 后，服务端先统计可索引文本的 Token：每 1,000 Token 20 点，最低 200 点。确认报价且成功建立索引后才结算；失败或取消会释放预留。15 天和 30 天会员每个生效周期包含 2 份知识材料额度。

会员额度针对成功索引的知识材料，不是空知识库集合。替换文件内容会形成新版本并重新报价；删除资料会立即停止未来检索，历史回答只保留最小来源名称和版本。
